



# 0000 Audi - Site Web

## Description du projet

Site web multipages dédié à la marque automobile Audi, réalisé dans le cadre de la SAÉ14

Le site web présente l'histoire d'Audi, ses modèles iconiques et les innovations technologiques qui ont fait la réputation de la marque allemande

## Lien vers le site web :

https://s5m87.github.io/SAE1.4/

## Différentes pages du site

### 0. Page de bienvenue (accueil.html)
- Animation avec son / possibilité de couper le son

### 1. Accueil (home.html)
- Titre et slogan
- Section "Pourquoi Audi ?" avec 3 avantages majeures
- Présentation des modèles iconiques

### 2. Histoire & Modèles (histoire.html)
- Chronologie de l'histoire d'Audi (1909 - aujourd'hui)
- Présentation des modèles iconiques (RS6 / R8 / e-tron GT / Quattro)
- Technologies et innovations (Quattro / MMI / éclairage LED)

### 3. À propos (apropos.html)
Page de présentation présentant
- Le projet en cours
- Admiration pour Audi et l'automobile
- Motivations derrière le projet

## Outils utilisées

- HTML - Corps de la page
- CSS - Mise en forme et animations
- Flexbox - Disposition flexible des éléments
- CSS Grid - Grilles de mise en page
- Media Queries - Design responsive
- Git & GitHub - Gestion de version
- GitHub Pages - Hébergement du projet (accessible publiquement)
